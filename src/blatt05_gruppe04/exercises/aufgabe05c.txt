Aufgabe 5c

Gewichteter Mittelwert
    Vorteile:
        - Liegt eine Partikelmenge vor, welche ziemlich nah beieinander liegt, so ist der Mittelwert dieser eine gute und stabile Poseschätzung

    Nachteile:
        - Häufig haben viele Partikel kein konstantes Gewicht und verändern sich regelmäßig. Wird nun immer der Mittelwert dieser genommen, so liegt in der Poseschätzung immer eine gewisse Art von "zittern" vor
        - Nimmt man den Mittelwert von zwei (oder mehr) sehr wahrscheinlichen Partikeln, welche aber weit auseinander liegen, so erhält man eine Poseschätzung, welche zwar das Mittel dieser ist, aber selbst keine wahrscheinliche Pose ist
        - Der Mittelwert von einigen validen Positionen, könnte selbst eine invalide (also z. B. in der Wand liegende) Position sein


Partikel mit bestem Gewicht:
    Vorteile:
        - Das Partikel mit dem höchsten Gewicht ändert sich meist nicht all zu häufig, was dazu führt, dass im Allgemeinen eine stabile Poseschätzung vorliegt

    Nachteile:
        - Liegen mehrere Partikel mit einem starken Gewicht vor, so wird nur eins von diesen ausgewählt und die anderen mehr oder weniger ignoriert
        - Liegt eine Partikelmenge vor, welche ziemlich nah beieinander liegt, so könnte es dazu kommen, dass all deren Gewichte ziemlich ähnlich sind und sich somit das Partikel mit dem höchsten Gewicht immer wieder verändert. Dadurch könnte dann eine gewisse Art von "zittern" entstehen
